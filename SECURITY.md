# Security Policy

## Supported Versions

This is a portfolio project. The latest release (`v0.1.0`) is the only
supported version.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security issues privately via GitHub's
[private vulnerability reporting](https://github.com/michalskimm/carbon-aware-mcp/security/advisories/new)
rather than opening a public issue. I'll respond as soon as I'm able.
